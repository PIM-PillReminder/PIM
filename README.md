# PIM
PIM(iOS)_Pill Reminder App

![pimimage](https://github.com/PIM-PillReminder/PIM/assets/88757043/235900d4-0c5b-4381-bc81-b8486c5a771b)


> 잊을리 없는 피임약 복용,‘PIM 핌’
---
### 핌 (PIM)
#### 피임약을 복용하는 여성들이 매일 같은 시간에 약을 복용하는 것을 잊지 않게 도와주자!

### 우리의 방향성

- ‘피임약’이라는 주제에만 **집중**한 서비스
- **귀찮음 극복**을 위한 기능
- **잊어버리지 않기** 위한 기능
